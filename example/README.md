### Build the Library

From the root of the library (dynamic-grid/):

```
npm i
npm run build
npm link
```

### Link It Inside the Example App

Now connect the example app to the linked library:

```
cd ../example
npm link ynotsoft-dynamic-grid
```

### Start the Example App

```
npm run dev
```
